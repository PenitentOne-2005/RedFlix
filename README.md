# 🎬 RedFlix

Мобильное приложение онлайн-кинотеатра на React Native.

## Tech Stack

- **React Native** — мобильная разработка
- **TypeScript** — типизация
- **React Navigation** — навигация между экранами

## Реализовано

- [x] Авторизация пользователей
- [x] Навигационная структура приложения

## В разработке

- [ ] Главный экран с каталогом фильмов
- [ ] Детальная страница фильма
- [ ] Поиск и фильтрация
- [ ] Избранное

## Запуск

```bash
# Установить зависимости
npm install

# Запустить на iOS
npx react-native run-ios

# Запустить на Android
npx react-native run-android
```
